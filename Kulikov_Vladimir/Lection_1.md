## 1. Что такое информатика

**Информатика** — наука об общих свойствах и структуре научной информации, а также о закономерностях её:
- создания,
- преобразования,
- накопления,
- передачи,
- использования.

---

## 2. Этимология термина «информатика»

Ключевые вехи:
- **XIV век:** *Information*
- **1955:** *Information Science* — сбор, классификация, хранение, поиск и распространение зафиксированного знания (как чистая и прикладная наука)
- **1955:** *Computer Science* — наука о теории вычислений и/или проектировании компьютеров
- **1957:** **Informatik** (нем.) — K. Steinbuch: *Informatik: Automatische Informationsverarbeitung*
- **1962:** *Informatique* (фр.) — Ph. Dreyfus (“information” + “automatique”)
- **1962:** «Информатика» (рус.) — А. А. Харкевич, Ф. Е. Темников

---

## 3. Полисемия слова «информатика»

Термин используют в разных смыслах:
- теория научно-информационной деятельности;
- наука о вычислительных машинах и их применении;
- фундаментальная наука об информационных процессах в природе, обществе и технических системах.

> Идея: многозначность термина может мешать пониманию, если не уточнять контекст.

## 5. Цели обучения (ожидаемые результаты)

После курса:
- понимать простую программу на C++;
- уметь написать простую программу на C++;
- получить навык программирования микроконтроллеров (**Arduino**);
- иметь общее понимание:
  - архитектуры и принципа работы компьютеров,
  - устройства ОС (на примере **Linux**),
  - устройства сетей передачи данных,
  - функционала баз данных;
- уметь писать простые запросы **SQL**;
- знать базовую терминологию компьютерных наук.

## 6. Язык C++: зачем изучать

Почему C++:
- один из самых распространённых синтаксисов;
- C и C++ — ключевые языки для **системного программирования**;
- C и C++ — важны для **микроконтроллеров**;
- существует формальный **стандарт языка**;
- хорошая **обратная совместимость**;
- огромная кодовая база (C — с 1972, C++ — с 1983);
- **мультипарадигменный** язык;
- несколько независимых компиляторов (в т.ч. open-source).

## 8. Среда разработки (IDE)

**IDE (Integrated Development Environment)** — интегрированная среда разработки: набор аппаратных и программных средств под единым интерфейсом (часто графическим) для поддержки разработки программ.

### Рекомендуемые IDE (общего назначения)
- Visual Studio (Windows, проприетарная, с 1997)
- VS Code (Windows/Linux/macOS, проприетарная, с 2015)
- GNU Emacs (Linux/Windows/macOS, GPL, с 1985)
- Vim (Linux/Windows/macOS, GPL, с 1992)

### Специализированные (Arduino)
- Arduino Legacy IDE (GPL, с 2005)
- Arduino IDE 2.0 (GPL, с 2019)
