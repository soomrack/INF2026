## 1) Память: базовые единицы информации

### Бит
- **бит** — двоичная цифра; сигнал или часть информации с двумя допустимыми значениями: **0** и **1**.
- **bit (binary digit)** — одна из цифр 0 или 1 в двоичной системе счисления.

---

### Байт
- **байт** — строка из нескольких битов, обрабатываемая как единое целое, обычно представляющая **знак** или часть знака.
- **byte** — string of bits, treated as a unit, usually representing a character or part of a character.

---

## 2) Бит и байт в стандарте C

Термины (ISO/IEC 9899:2024, язык C):
- **bit** — единица хранения данных в среде выполнения, достаточная для хранения объекта с одним из двух значений; адресовать каждый отдельный бит объекта может быть невозможно.
- **byte** — адресуемая единица хранения данных, достаточная, чтобы хранить любой символ базового набора символов среды выполнения.
- **low-order bit** — младший (наименее значимый) бит в байте.
- **high-order bit** — старший (наиболее значимый) бит в байте.

---

## 3) Байт в стандарте C++

Термины (ISO/IEC 14882:2024, язык C++):
- **byte** — фундаментальная единица хранения в модели памяти C++.
- Байт:
  - достаточно велик, чтобы содержать кодировку любого элемента **basic literal character set**,
  - и 8-битные кодовые единицы **UTF-8**,
  - состоит из непрерывной последовательности битов, число которых **implementation-defined** (зависит от реализации).
- Память, доступная C++ программе, состоит из одной или нескольких последовательностей **смежных байтов**.
- Каждый байт имеет **уникальный адрес**.

---

## 4) Октет

- **октет** — упорядоченная последовательность из **8 бит**, единица измерения количества информации, эквивалентная **8-битовому байту**.
- **octet (8-bit byte)** — byte that consists of eight bits.

---

## 5) Приставки СИ (десятичные)

Приставки для **байтов** и **битов** (основание 10):

- 10³ **кило-** (к): килобайт (кБ), килобит (кб)
- 10⁶ **мега-** (М): мегабайт (МБ), мегабит (Мб)
- 10⁹ **гига-** (Г): гигабайт (ГБ), гигабит (Гб)
- 10¹² **тера-** (Т): терабайт (ТБ), терабит (Тб)
- 10¹⁵ **пета-** (П): петабайт (ПБ), петабит (Пб)
- 10¹⁸ **экса-** (Э): эксабайт (ЭБ), эксабит (Эб)
- 10²¹ **зетта-** (З): зеттабайт (ЗБ), зеттабит (Зб)
- 10²⁴ **йотта-** (И): йоттабайт (ИБ), йоттабит (Иб)

Примечания со слайда:
- *эксафлоп* — суперкомпьютер Frontier (2022)
- *зеттабайт* — годовой объём интернет-трафика (2016)

### Годы введения приставок СИ (из лекции)

## 7) Системы счисления

**Позиционная система счисления** — система, где значение цифры зависит от её позиции в записи числа.

Наборы цифр:
- десятичные: 0–9
- шестнадцатеричные: 0–9, A–F
- восьмеричные: 0–7
- двоичные: 0–1

---

## 8) Запись чисел в C++

Пример одного и того же числа (13) в разных системах:
```cpp
int a = 13;         // десятичная
int b = 0x0D;       // шестнадцатеричная (0x...)
int c = 015;        // восьмеричная (0...)
int d = 0b00001101; // двоичная (0b...)
````

Битовые сдвиги:

```cpp
d >> 2; // 0b00000011
d << 1; // 0b00011010
```

## 9) Порядок байтов (Endianness)

Вопрос со слайда:

> `00000001 00000000` — это число **1** или число **256**?

Определения:

* **big-endian** — порядок от старшего к младшему (сетевой порядок TCP/IP)
* **little-endian** — порядок от младшего к старшему (x86, USB)
* **bi-endian** — переключаемый порядок (ARM)
* **middle-endian / mixed-endian** — смешанный порядок байтов

## 10) Адрес переменной: указатели и ссылки

```cpp
int *a;      // a — указатель на int
int* b, c;   // b — указатель, c — int

void *d;     // указатель "на что угодно" (тип не задан)

a = &c;      // a указывает на данные переменной c
*a = 3;      // теперь c = 3

int& e = c;  // e — псевдоним (ссылка) на переменную c

a = nullptr; // a равно 0 — C++
a = NULL;    // a равно 0 — C
```

## 11) Арифметика указателей

```cpp
int a[10];   // массив из 10 int
// a — указатель на начало массива

a[0] = 10;   // индексация с 0
a[9] = 19;

int *b = a;  // b указывает на нулевой элемент массива
b += 1;      // теперь b указывает на первый элемент массива
// значение b увеличилось на sizeof(int)

b[2] == *(b + 2); // [x] — это прибавление x к указателю и разыменование
```
