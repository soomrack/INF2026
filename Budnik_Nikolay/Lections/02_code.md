# Лекция 02. Программный код и базовый синтаксис C++

## Кратко о лекции

Лекция объясняет, что такое программа, какие бывают парадигмы программирования, из каких элементов состоит код и какие базовые конструкции используются в C++

## 1. Что такое программа

Программа — это набор данных и инструкций, которые управляют работой вычислительной системы и реализуют определённый алгоритм

Проще: программа говорит компьютеру, что нужно сделать и в каком порядке

## 2. Императивное программирование

Императивное программирование описывает способ получения результата

Характерные признаки:

- программа состоит из инструкций
- инструкции выполняются последовательно
- данные читаются из памяти
- результат работы инструкций записывается в память

Примеры языков:

- C
- C++
- Pascal
- JavaScript
- Java
- Python

Главная идея: программист подробно указывает шаги выполнения

## 3. Декларативное программирование

Декларативное программирование описывает ожидаемый результат, а не пошаговый способ его получения

Примеры:

- HTML
- LaTeX
- SQL

Главная идея: программист задаёт, что должно получиться, а система сама выбирает способ выполнения

## 4. Базовые элементы программы

Основные элементы программы:

- переменные
- инструкции
- выражения
- блоки кода

Типичные инструкции:

- присваивание результата выражения
- безусловный переход
- условный переход
- вызов функции

Блок — это группа инструкций, которые выполняются как единая часть программы

## 5. Три конструкции управления

По теореме Бёма — Якопини любой алгоритм можно представить через три конструкции

### 5.1 Последовательное исполнение

Команды выполняются одна за другой

```cpp
{
    blockA;
}
{
    blockB;
}
```

### 5.2 Ветвление

Программа выбирает один из вариантов в зависимости от условия

```cpp
if (condition) {
    blockB;
} else {
    blockC;
}
```

### 5.3 Цикл

Один и тот же блок повторяется, пока условие истинно

```cpp
while (condition) {
    blockB;
}
```

## 6. Структурное программирование

Структурное программирование строит программу из вложенных конструкций с одной точкой входа и одной точкой выхода

Зачем это нужно:

- код легче читать
- проще искать ошибки
- проще доказывать правильность алгоритма
- меньше хаоса от `goto`

Основная идея: программу лучше строить из понятных блоков, а не из произвольных переходов

## 7. Схемы алгоритмов

В быту часто говорят «блок-схема», но в стандартах ЕСПД используются термины:

- схема программы
- схема алгоритма

Схема помогает показать логику программы графически: начало, ввод данных, проверку условий, действия и завершение

## 8. Переменные

Переменная — это именованный объект, который хранит значение

Пример:

```cpp
int age = 18;
```

Здесь:

- `int` — тип данных
- `age` — имя переменной
- `18` — значение

Переменная нужна, чтобы программа могла сохранять и изменять данные

## 9. Целочисленные типы

Примеры целочисленных типов в C++:

```cpp
int a;
short int b;
long int c;
long long int d;
unsigned int e;
char h;
```

`unsigned` означает, что число не может быть отрицательным

`char` часто используется для символов, но технически тоже хранит числовое значение

## 10. Вещественные и логические типы

Вещественные типы:

```cpp
float a;
double b;
long double c;
```

Логический тип:

```cpp
bool flag;
```

`bool` принимает только два значения:

- `true`
- `false`

## 11. Пользовательские типы

В C++ можно создавать собственные типы

```cpp
using USD = int;

struct Account {
    USD money;
    float interest;
};
```

`struct` позволяет объединить несколько полей в один объект

## 12. Цикл while

`while` выполняет блок, пока условие истинно

```cpp
int k = 0;

while (k < 100) {
    k++;
}
```

Команды `continue` и `break` управляют ходом цикла:

- `continue` пропускает оставшуюся часть текущего шага
- `break` полностью завершает цикл

## 13. Цикл for

`for` удобен, когда заранее понятен счётчик цикла

```cpp
for (int k = 0; k < 100; ++k) {
    b++;
}
```

Обычно `for` используют для перебора чисел, массивов и повторяющихся действий

## 14. Цикл do while

`do while` сначала выполняет тело цикла, а потом проверяет условие

```cpp
int k = 0;

do {
    k++;
} while (k < 100);
```

Такой цикл выполнится хотя бы один раз

## 15. Условный оператор

`if else` выбирает действие по условию

```cpp
if (a > 100) {
    k++;
} else {
    k--;
}
```

Если условие истинно, выполняется первый блок. Иначе выполняется блок после `else`

## 16. Функции

Функция — это отдельный блок кода, который можно вызывать по имени

```cpp
int sum(int a, int b)
{
    int c = a + b;
    return c;
}
```

Функции нужны, чтобы не повторять код и делить программу на понятные части

## 17. Область видимости

Область видимости переменной заканчивается ближайшими фигурными скобками, внутри которых она была объявлена

```cpp
int main()
{
    int a = 0;

    for (int k = 0; k < 100; ++k) {
        int b = 0;
        a++;
    }

    // b здесь уже недоступна
}
```

Переменная уничтожается при выходе из своей области видимости

## Итог лекции

Программа строится из переменных, инструкций, условий, циклов и функций. Главная задача на этом этапе — научиться читать порядок выполнения кода и понимать, где живут данные
