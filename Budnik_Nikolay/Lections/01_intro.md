# Лекция 01. Введение в информатику, C++ и среду разработки

## Кратко о лекции

Лекция вводит курс информатики: объясняет, что изучает информатика, почему в курсе используется C++, какие инструменты нужны для работы и что нужно подготовить к практике

## 1. Что такое информатика

Информатика изучает информацию и процессы работы с ней:

- создание
- преобразование
- накопление
- хранение
- передачу
- использование

В курсе информатика понимается широко: это не только программирование, но и устройство компьютеров, операционных систем, сетей и баз данных

## 2. Происхождение термина

Слово «информатика» связано с несколькими направлениями:

- **Information Science** — сбор, классификация, хранение и распространение знаний
- **Computer Science** — теория вычислений и устройство компьютеров
- **Informatik** — автоматическая обработка информации
- **Informatique** — соединение слов `information` и `automatique`

Главная идея термина: информация должна не просто существовать, а обрабатываться, передаваться и использоваться с помощью технических средств

## 3. Почему термин может пониматься по-разному

У слова «информатика» есть несколько значений:

- теория научно-информационной деятельности
- наука о вычислительных машинах
- наука об информационных процессах в природе, обществе и технике

Поэтому в одном контексте информатика может означать программирование, а в другом — более общую науку об информации

## 4. План курса

Основные темы курса:

1. язык C++
2. архитектура компьютера
3. операционные системы
4. сети передачи данных
5. базы данных

Логика курса простая: сначала изучается код, затем устройство компьютера, потом более крупные программные и сетевые системы

## 5. Цели обучения

После курса нужно уметь:

- читать простые программы на C++
- писать простые программы на C++
- программировать Arduino
- понимать общую архитектуру компьютера
- понимать базовые принципы Linux
- понимать устройство сетей передачи данных
- понимать назначение баз данных
- писать простые SQL-запросы
- использовать базовую терминологию компьютерных наук

## 6. Почему изучается C++

C++ выбран потому, что он важен для системного программирования и микроконтроллеров

Причины изучать C++:

- синтаксис C/C++ повлиял на многие языки
- C и C++ применяются в системном программировании
- C и C++ часто используются для микроконтроллеров
- язык стандартизирован
- есть обратная совместимость
- существует много старого и нового кода
- язык поддерживает разные стили программирования
- есть несколько независимых компиляторов

C++ сложнее многих учебных языков, зато хорошо показывает связь программы с памятью, компилятором и железом

## 7. Стандарты C и C++

Языки C и C++ развиваются через стандарты ISO/IEC

Примеры стандартов C++:

- C++98
- C++03
- C++11
- C++14
- C++17
- C++20
- C++24

Примеры стандартов C:

- C89
- C90
- C99
- C11
- C18
- C24

Стандарт нужен, чтобы разные компиляторы одинаково понимали язык

## 8. Среда разработки

**IDE** — интегрированная среда разработки

Она объединяет инструменты для написания программ:

- редактор кода
- подсветку синтаксиса
- запуск компилятора
- отладку
- работу с файлами проекта

IDE нужна не для самого языка, а для удобной разработки

## 9. Рекомендуемые IDE

Среды общего назначения:

- Visual Studio
- Visual Studio Code
- GNU Emacs
- Vim

Среды для Arduino:

- Arduino Legacy IDE
- Arduino IDE 2.0

Для начала достаточно установить одну удобную IDE и научиться запускать в ней простую программу

## 10. Компиляторы

Компилятор переводит исходный код в программу, которую может выполнить компьютер

Примеры компиляторов:

- `g++` для Linux
- `clang` для Linux и macOS
- MSVC для Windows
- MinGW для Windows

Без компилятора код на C++ сам по себе не запускается

## 11. Первая программа

Пример программы `Hello World`:

```cpp
#include <stdio.h>

int main()
{
    printf("Hello World!");
}
```

Смысл программы:

- `#include <stdio.h>` подключает функции ввода-вывода
- `main()` — точка входа в программу
- `printf()` выводит текст на экран

## 12. Терминал

Терминал — средство для ввода команд и получения результата работы программ

Пример работы в терминале:

```text
gcc.exe hello.cpp
hello.exe
Hello World!
```

Терминал нужен, чтобы компилировать, запускать программы и работать с инструментами разработки

## 13. Git и GitHub

**Git** — распределённая система управления версиями

Git позволяет:

- хранить историю изменений
- смотреть различия между версиями
- возвращаться к старым версиям
- работать над проектом совместно

**GitHub** — платформа для хранения проектов и совместной разработки

## 14. Что подготовить к практике

К практическому занятию нужно:

- установить IDE
- установить компилятор
- зарегистрироваться на GitHub
- настроить среду разработки
- скомпилировать и запустить `Hello World`
