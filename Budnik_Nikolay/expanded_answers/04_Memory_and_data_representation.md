# Вопрос 27. Позиционная система счисления, двоичная, восьмеричная, шестнадцатеричная

## Полный экзаменационный ответ

Позиционная система счисления - это способ записи чисел, при котором значение каждой цифры зависит не только от самой цифры, но и от ее позиции в числе. Например, в десятичной записи число `123` означает не просто набор цифр `1`, `2`, `3`, а сумму `1 * 10^2 + 2 * 10^1 + 3 * 10^0`. Основание системы счисления показывает, сколько разных цифр используется и во сколько раз меняется вес соседних разрядов.

В информатике особенно важны двоичная, восьмеричная и шестнадцатеричная системы счисления. Двоичная система имеет основание 2 и использует только цифры `0` и `1`. Она напрямую связана с устройством компьютера, потому что бит тоже имеет два состояния: `0` или `1`. Поэтому любые данные в памяти в конечном счете можно рассматривать как последовательности двоичных разрядов.

Восьмеричная система имеет основание 8 и использует цифры от `0` до `7`. Она удобна тем, что одна восьмеричная цифра соответствует ровно трем битам. Сейчас она используется реже, но встречается в Unix/Linux-правах доступа и в исторической записи числовых литералов в C/C++.

Шестнадцатеричная система имеет основание 16 и использует цифры `0-9` и буквы `A-F`, где `A` означает 10, `B` - 11, `C` - 12, `D` - 13, `E` - 14, `F` - 15. Она особенно удобна для программистов, потому что одна шестнадцатеричная цифра соответствует четырем битам. Поэтому байт из 8 бит можно записать двумя hex-цифрами: например, `11111111_2 = FF_16`.

В C++ разные системы счисления имеют разные формы записи. Десятичное число пишется обычно, например `13`. Восьмеричный литерал исторически записывается с начального нуля, например `015`, и это означает десятичное `13`. Шестнадцатеричный литерал пишется с префиксом `0x`, например `0x0D`. Двоичный литерал в современном C++ можно записать с префиксом `0b`, например `0b1101`.

## Основные определения

- Позиционная система счисления - система записи чисел, где вес цифры зависит от ее позиции.
- Основание системы счисления - количество цифр, используемых в системе, и коэффициент перехода между разрядами.
- Двоичная система - система с основанием 2, использующая цифры `0` и `1`.
- Восьмеричная система - система с основанием 8, использующая цифры `0-7`.
- Шестнадцатеричная система - система с основанием 16, использующая `0-9` и `A-F`.

## Ключевые идеи

- Значение числа в позиционной системе получается как сумма цифр, умноженных на степени основания.
- Двоичная система естественна для компьютера, потому что соответствует битам.
- Hex-запись компактно представляет байты: 1 hex-цифра = 4 бита.
- Восьмеричная запись группирует биты по 3, шестнадцатеричная - по 4.
- В коде C++ префиксы `0`, `0x` и `0b` меняют основание числового литерала.

## Примеры

Перевод двоичного числа в десятичное:

```text
1101_2 = 1 * 2^3 + 1 * 2^2 + 0 * 2^1 + 1 * 2^0
       = 8 + 4 + 0 + 1
       = 13_10
```

Перевод десятичного числа `25` в двоичное через деление на 2:

```text
25 / 2 = 12, остаток 1
12 / 2 = 6,  остаток 0
6 / 2  = 3,  остаток 0
3 / 2  = 1,  остаток 1
1 / 2  = 0,  остаток 1

Остатки читаются снизу вверх: 25_10 = 11001_2
```

Связь двоичной и шестнадцатеричной записи:

```text
101111_2 = 0010 1111_2 = 2F_16
```

Пример C++-литералов:

```cpp
int a = 13;       // десятичная запись
int b = 015;      // восьмеричная запись, значение 13
int c = 0x0D;     // шестнадцатеричная запись, значение 13
int d = 0b1101;   // двоичная запись, значение 13
```

## Неочевидные моменты

- `012` в C/C++ означает не двенадцать, а восьмеричное число, равное `10` в десятичной системе.
- В математическом тексте основание часто пишут нижним индексом: `1010_2`, но в коде C++ используются префиксы: `0b1010`, `0x0A`.
- При переводе из десятичной системы в двоичную остатки от деления читаются снизу вверх.
- В шестнадцатеричной системе буквы `A-F` являются цифрами со значениями от 10 до 15.
- Двоичная, восьмеричная, десятичная и шестнадцатеричная записи могут обозначать одно и то же число.

## Краткий ответ

Позиционная система счисления - это система, где значение цифры зависит от ее позиции и основания системы. В информатике важны двоичная, восьмеричная и шестнадцатеричная системы, потому что они удобно связаны с битами. Двоичная система использует `0` и `1`, восьмеричная группирует биты по 3, а шестнадцатеричная - по 4. В C++ для разных оснований используются записи вроде `0b1101`, `015` и `0x0D`.

## Краткие вопросы для викторины

- Вопрос: Что означает основание системы счисления?
  Правильный краткий ответ: количество цифр и вес разрядов.
  Допустимые варианты ответа: количество цифр; основание задает веса разрядов; база системы; число разных цифр.
  Комментарий при ошибке: Основание определяет, какие цифры используются и как растут веса позиций.

- Вопрос: Сколько бит соответствует одной шестнадцатеричной цифре?
  Правильный краткий ответ: 4 бита.
  Допустимые варианты ответа: 4; четыре; четыре бита; nibble.
  Комментарий при ошибке: Одна hex-цифра имеет 16 вариантов, а 16 = 2^4, значит ей соответствуют 4 бита.

- Вопрос: Чему равно `1101_2` в десятичной системе?
  Правильный краткий ответ: 13.
  Допустимые варианты ответа: 13; тринадцать; 13_10.
  Комментарий при ошибке: Разложите число по степеням двойки: 8 + 4 + 1 = 13.

- Вопрос: Что означает литерал `0x0D` в C++?
  Правильный краткий ответ: шестнадцатеричное число 13.
  Допустимые варианты ответа: hex 13; 13; шестнадцатеричная запись числа 13; 0D в hex.
  Комментарий при ошибке: Префикс `0x` указывает на шестнадцатеричную систему счисления.
