# Блок 3. Basic C++ syntax

## Вопрос 18. Что такое переменная?

Переменная — это именованный объект в программе, предназначенный для хранения значения определённого типа. У переменной есть имя, тип, область видимости, время жизни, адрес в памяти и текущее значение.

Например, в записи `int a = 10;` имя переменной — `a`, тип — `int`, а начальное значение — `10`. Тип определяет, какие значения можно хранить и какие операции допустимы. Значение обычной переменной может изменяться во время выполнения программы, но если переменная объявлена как `const`, её значение после инициализации менять нельзя.

## Вопрос 19. Базовые типы данных

Базовые типы данных в C++ — это встроенные типы, из которых строятся более сложные структуры. К ним относятся целочисленные типы `short`, `int`, `long`, `long long`, символьный тип `char`, логический тип `bool`, вещественные типы `float`, `double`, `long double`, а также специальный тип `void`.

Целочисленные типы могут быть знаковыми (`signed`) и беззнаковыми (`unsigned`). `bool` хранит логическое значение `true` или `false`. `char` обычно используется для хранения символа или небольшого целого значения. Вещественные типы нужны для чисел с дробной частью. Размеры типов могут зависеть от платформы и компилятора, поэтому при переносимом коде нужно быть аккуратным.

## Вопрос 20. Пользовательские типы данных

Пользовательские типы данных — это типы, созданные самим программистом на основе возможностей языка. Они нужны, чтобы описывать объекты предметной области более понятно и безопасно, чем просто набором стандартных типов.

В C++ пользовательские типы можно создавать с помощью `struct`, `class`, `enum`, `typedef` и `using`. Например, структура `Account` может объединять баланс, процентную ставку и методы работы со счётом. Класс позволяет не только хранить данные, но и связать с ними поведение, а перечисление (`enum`) удобно для ограниченного набора значений, например режимов работы устройства.

## Вопрос 21. Циклы

Цикл — это управляющая конструкция, которая многократно выполняет блок кода. В C++ основные циклы — `while`, `for` и `do while`.

Цикл `while` сначала проверяет условие и выполняет тело, пока оно истинно. Цикл `for` удобен, когда есть счётчик или заранее понятная схема изменения переменной. Цикл `do while` сначала выполняет тело, а потом проверяет условие, поэтому он выполнится хотя бы один раз. Внутри циклов можно использовать `break`, чтобы полностью выйти из цикла, и `continue`, чтобы перейти к следующей итерации.

## Вопрос 22. Условные операторы

Условные операторы позволяют выполнять разные участки кода в зависимости от логического условия. Основная конструкция в C++ — `if`, которая может использоваться вместе с `else` и `else if`.

```cpp
if (a > 100) {
    k++;
} else {
    k--;
}
```

Если условие в `if` истинно, выполняется первый блок; если ложно — блок после `else`. Условные операторы позволяют программе реагировать на данные и принимать решения. Важно помнить, что условие должно иметь логический смысл, а блоки в фигурных скобках образуют отдельную область видимости.

## Вопрос 23. Функции

Функция — это именованный блок кода, выполняющий отдельную задачу. У функции есть тип возвращаемого значения, имя, список параметров и тело.

```cpp
int sum(int a, int b)
{
    int c = a + b;
    return c;
}
```

В этом примере функция `sum` принимает два параметра типа `int`, складывает их и возвращает результат. Функции нужны, чтобы не повторять один и тот же код, делить программу на понятные части и скрывать детали реализации. Если функция ничего не возвращает, её типом указывают `void`.

## Вопрос 24. Область видимости

Область видимости определяет, где в программе можно обращаться к имени переменной, функции или другого объекта. В C++ локальная область видимости обычно ограничивается ближайшими фигурными скобками, внутри которых объект объявлен.

```cpp
int main()
{
    int a = 0;

    for (int k = 0; k < 10; ++k) {
        int b = 1;
        a += b;
    }

    return 0;
}
```

В этом примере переменная `a` доступна почти во всей функции `main`, переменная `k` доступна внутри цикла `for`, а переменная `b` — только внутри тела цикла. После выхода из области видимости локальная переменная уничтожается, и обращаться к ней нельзя.
