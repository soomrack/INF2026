

* Задание 02. "Умная теплица"

Исходные данные

Конструкция умной теплицы состоящая из
- Датчики:
  1. температура воздуха
  2. влажность воздуха
  3. влажность почвы
  4. освещенность
- Актуаторы:
  1. вентилятор
  2. нагреватель
  3. лампа
  4. помпа
- Микроконтроллер
  1. Arduino UNO v3

Требуемый функционал:
  Программа должна поддерживать микроклимат в теплице для выращивания растения.

Уточнение (регулирование):
  1. при недостатке освещения лампа должна включаться, за исключением ночного времени
  2. при сухости почвы должен быть полив (помпа)
  3. при повышенной температуре нужно проветривать
  4. при пониженной температуре нужно включать обогреватель и вентилятор
  5. при повышенной влажности нужно проветривать (опционально, повысить температуру)
  6. проветривание по расписанию (кроме ночного времени)

Уточнение (программа):
  1. глядя на маленький кусок программы (обычно основный цикл) должно быть понятно:
     + какие актуаторы есть в теплице
     + какие датчики есть в теплице
     + функционал программы (что регулируется, контролируется, ...)
  2. возможность добавления новых датчиков не меняя архитектуры программы
  3. возможность добавления новых актуаторов не меняя архитектуры программы
  4. возможность добавления новых алгоритмов в рамках выбранной архитектуры
  5. программа должна позволять легко заменять используемые датчики и актуаторы в алгоритмах регулирования
  6. программа должна позволять легко переключаться с поддержания одного климата на другой


При написании программы нужно использовать Open-Closed принцип.
